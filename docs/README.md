# `crn-elements`

> 用于 React Native 的组件库，该组件库使用 Typescript 开发，提供了更方便的类型支持。

### 安装

```bash
# 使用npm
npm install --save @codoonfxd/crn-elements

# 使用yarn
yarn add @codoonfxd/crn-elements
```

### 使用

```javascript
import { Button } from '@codoonfxd/crn-elements';

export default () => {
  return <Button title="按钮" />;
};
```
